

## 🧭 页面名称：店铺信息弹窗（Store Information Modal）

### 📐 页面类型

* **组件类型**：Modal / Bottom Sheet（底部弹出浮层）
* **展示形式**：自下向上弹出，半透明遮罩层背景
* **圆角样式**：顶部圆角约 16px
* **背景色**：#FFFFFF
* **关闭方式**：右上角灰色圆形「×」按钮

---

## 💡 交互说明

| 交互元素    | 动作 | 行为结果           |
| ------- | -- | -------------- |
| “×” 按钮  | 点击 | 关闭弹窗，返回主页面     |
| “电话” 按钮 | 点击 | 调用微信拨号功能       |
| “到店” 按钮 | 点击 | 打开微信地图并导航到门店位置 |
| 背景区域    | 点击 | 关闭弹窗（遮罩可点击关闭）  |

---

## ✨ 视觉与交互风格总结

| 项目   | 设计特点            |
| ---- | --------------- |
| 色彩   | 中性灰+品牌强调色（橙/蓝）  |
| 字体层级 | 标题>小节标题>正文      |
| 信息密度 | 低，留白充足，强调可读性    |
| 动画   | 自下而上弹出 + 渐显遮罩   |
| 可访问性 | 字体对比清晰，操作按钮间距合理 |
